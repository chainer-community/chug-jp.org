    chug: Chainer User Group¶

    chug (Chainer User Group)は、有志による Chainer のコミュニティです。

    chug は主に以下の活動を行っています。

    • Chainer Meetup 等のイベント活動
    • chainer-colab-notebookの作成

    イベント¶

    Chainer Meetup #08 開催予定です！ https://chainer.connpass.com/event/106292/

    コンテンツ¶

    Awsome Chainer

    Awsome Chainer では、Chainerに関連するリンクを集めています。

    Chainer Colab Notebook

    Chainer Colab Notebook では、Google Colaboratory上で動作するChainerのノートブックを集めています。 Google Colaboratory は、無料で使えるJupyter Notebook環境です。様々な制限はありますが、 GPUも無料で使えるのが魅力です。

    Chainer Colab Notebookは、誰でも簡単にChainerの学習をはじめられるように、Google Colaboratory上で 動作するChainerのチュートリアル・サンプル等を集めています。
